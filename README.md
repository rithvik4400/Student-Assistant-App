# 📚 Student Assist App

## 🚀 Overview

Student Assist is an Android application designed to help students manage their academic activities efficiently. The app provides features like notes management, task tracking, and productivity tools to enhance student learning and organization.

---

## ✨ Features

* 📒 Create, edit, and manage notes
* 🧠 AI-based doubt solving assistance
* 📊 Student productivity dashboard
* 🔐 User authentication (Login/Signup)
* ⏰ Task and reminder management
* 📱 Clean and user-friendly UI (Jetpack Compose)

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **Framework:** Android (Jetpack Compose)
* **Backend:** Firebase Authentication & Firestore
* **IDE:** Android Studio
